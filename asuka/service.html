asuka.service — Service interface¶

class asuka.service.Service(name, config={}, required_apt_packages=frozenset([]), required_python_packages=frozenset([]))¶

The inteface of services.

Parameters:
  • name (basestring) – the service name
  • config (collections.Mapping) – the config mapping object
  • required_apt_packages (collections.Set) – the set of APT packages the service depends
  • required_python_packages (collections.Set) – the set of Python packages the service depends. elements have to be PyPI names
NAME_PATTERN = <SRE_Pattern object at 0x000000010791a220>¶

(re.RegexObject) The pattern of the valid service name.

install(instance)¶

Installs the service into the instance.

Parameters:instance (asuka.instance.Instance) – the instance to install the service
required_apt_packages¶

(collections.Set) The set of APT package names to install e.g. frozenset(['python-dev', 'python-greenlet']).

required_python_packages¶

(collections.Set) The set of PyPI package names to install e.g. frozenset(['Werkzeug', 'chardet']).
