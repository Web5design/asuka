asuka.app — Application configuration¶

class asuka.app.App(**values)¶

Application configuration. It takes keyword-only parameters that are the same name of settable properties and attributes.

KEY_PAIR_NAME_FORMAT = 'Asuka-{app.name}'¶

(basestring) The format string of AWS key pair name.

config_dir = 'asuka/'¶

(basestring) The name of the Asuka configuration directory in the repository. The default uses asuka/.

ec2_connection = None¶

(boto.ec2.connection.EC2Connection) The EC2 connection to invoke APIs.

github¶

(github3.GitHub) The GitHub connection.

key_name¶

(basestring) The human-readable title of the key pair.

key_pair¶

(boto.ec2.keypair.KeyPair) The EC2 key pair matched to private_key.

name = None¶

(basestring) The name of the app.

private_key¶

(paramiko.pkey.PKey) The pair of public and private key.

public_key_string¶

(basestring) The public key string.

repository¶

(github3.repos.Repository) The repository of the app.
